Parameters influencing the oscillations


Several parameters determine the actual behavior of oscillating systems. Minute differences can lead to a completely different behavior. Thus, the understanding if these dependencies is crucial for a decoding of the shell patterns.

To trigger the next activation, a baseline activation is required from which the autocatalysis can start. In the simulation at right, the basic activator production is lower, ba = 0.04 instead of 0.1 at left. Therefore, the inhibitor has to drop to a lower level before the next activation can start. The avalanche-like increase is much stronger, the peak height  higher, and the oscillation frequency is lower since it takes longer until the next trigger occurs.




Other way around, a baseline inhibitor production (bb in Eq. 2.1) can suppress the onset of the next activation and the system will be sleeping. It remains in a non-activated steady state until a trigger occurs. Such cells are excitable. The trigger can be caused by some spread from an activated neighboring cell. In an extended field, such mode can lead to traveling waves if one cell activates the next. After surpassing a certain threshold, each cell goes through the full cycle. Some exchange with neighboring cells cause their trigger. Due to the accumulating inhibitor, the cells return to the resting state. As long as this inhibitor is still high, the cells are in a refractory state, i.e., insensitive to a further trigger.
